卡特琳娜·斯福尔扎（Caterina Sforza）教廷国务卿，米兰公爵。也是Ax的实际负责人。
《圣魔之血》人物之一。
卡特琳娜·斯福尔扎，有“铁女”之称的米兰大公、枢机卿、教皇厅国务圣省长官。教皇厅国务圣省特务分室（通称Ax）是其直属机构。身着红衣，外表美丽，性格干练，处事果断，对异母弟弟呵护有加（不知道是不是因为她是保守派的关系），是个不折不扣的女强人。
当年该隐觉醒后，派人去刺杀自己的弟弟亚伯。位置恰好就是斯福尔扎家族的祖宅。结果斯福尔扎族被灭门，她孤身逃入地下墓穴，与亚伯见面。从此立下了对抗世界公敌的决心。之后她自原先就读的法学院转去罗马就读神学，利用自己身为教皇私生女的身份逐渐培养势力，终于成为教廷群臣中的青年才俊。
正当此时，教皇去世了，继任者没有决定。最热门人选是教皇的弟弟，当时异端审判局局长艾方索大主教。但是此人虽然能力出众，却满脑子狂热宗教思想，手段之激进连同为鹰派的美第奇也为之心寒。于是，卡特琳娜和美第奇这对同父异母兄妹第一次也是最后一次联手，利用各自掌握的关系力捧他们的弟弟，教皇的嫡生子亚历山大。经过一番操作，成功地让亚历山大继任教皇，并将艾方索流放至科隆。至此，卡特琳娜—美第奇的政治格局正式确立。
虽然卡特琳娜的政治才能出众，但在军事方面却反应不佳，多次被美第奇的军事行动将一军。好在身边能人众多，最终总能化险为夷。
后来患有胶原性疾病，生命垂危，时常咳血。而且似乎对亚伯有着不一般的感情，疑似喜爱。
